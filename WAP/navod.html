TimeLine

By Matej Marusak, created as school project on BUT, 2018

What is this

Draw timeline from timestaps in your html. Toggle between real timestamps and relative ones.

Features

  • Draw vertical line with all timestamps extracted from html
  • Toggle between real and relative timestamp
  • Automatically recalculate relative timestamps and update them
  • Events are spread according to time in which they happened (events that happened closer to each other are closer on the timeline)
  • Group events into one which are too close (and show only hint what is inside)
  • On hover of timepoint show popup with all items with full text

How to use

Include script


        <script src="script.js" type="text/javascript"/></script>

Call Main function


    <script>
        Main();
   </script>

Alternatively you can uncomment 16th line in script, to load automatically

Style to fit your site

  • Format .controls to position the whole timeline
  • Format #timeStampToggle to set style of the toggle button
